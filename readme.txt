Smart recent posts shows the recent posts from your wordpress blog

== Description ==

This plugin contains a widget for showing off the recent posts from your wordpress blog.

With this plugin you can choose number of posts to display, display featured image, dimensions of featured image, posts order, select the categories also. We will adding more options in the next version.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `smart-recent-post` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Head over the the widgets settings page and enable `Smart Recent Post`


== Frequently Asked Questions ==

= How to use it? =
Enable the plugin and place the widget in the sidebar (where you wants)
